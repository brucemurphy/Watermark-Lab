# Watermark-Lab
Watermark Lab is a lightweight Windows utility that adds tiled, diagonal, semi‑transparent watermarks to both PowerPoint files and video formats. It uses a Tkinter dark‑theme GUI, automates PowerPoint via COM for native slide edits, and relies on ffmpeg plus a Pillow‑generated overlay for video processing.
